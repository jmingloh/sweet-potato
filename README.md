## Recommended GitHub Setup

Create a dedicated repository for this website, for example:
- `sweet-potato`

Then upload only files from this folder:
- `index.html`
- `index-cn.html`
- `index-ms.html`
- related `.html`, `.js`, `.css`, images, `robots.txt`, `sitemap.xml`

## After Upload

1. Enable GitHub Pages for the repo
2. Confirm the site loads correctly
3. Submit the sitemap to Google Search Console:
   - `https://jmingloh.github.io/sweet-potato/sitemap.xml`
4. Request indexing for:
   - home page
   - Chinese page
   - Malay page
   - order / group-buy / agent pages
